CHANGE

This command is no longer available in the menus, but is retained for use in the command line. The command allows the user to change previously entered information about the specimen being identified. See also USE, DELETE.

Syntax of command line: CHANGE c 1, v 1 c 2, v 2 ... where c 1, c 2, ... are character numbers, ranges, or keywords, and v 1, v 2, ... are state values (or numerical or text values, depending on the character type). The state values and the preceding commas are optional; if they are not specified, the program prompts for them.
