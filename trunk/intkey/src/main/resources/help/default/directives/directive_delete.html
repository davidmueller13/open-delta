DELETE

This command is no longer available in the menus, but is retained for use in the command line. The command allows the user to remove previously entered information about the specimen being identified. See also CHANGE, USE.

Syntax of command line: DELETE switch c 1 c 2 ...

where c 1, c 2 ... are character numbers, ranges or keywords. The optional switch /M suppresses the warning message when unused characters are specified.
