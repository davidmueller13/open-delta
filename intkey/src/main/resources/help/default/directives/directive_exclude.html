EXCLUDE

This command is used to exclude characters or taxa from consideration by other commands. It is equivalent to an INCLUDE command specifying those characters or taxa not specified in the EXCLUDE command. See INCLUDE for more details.

Syntax of command line:
EXCLUDE CHARACTERS c1 c2 ...
EXCLUDE TAXA t1 t2 ...

where c1, c2 ... are character numbers, ranges or keywords, and t1, t2 ... are taxon numbers, ranges or keywords.
