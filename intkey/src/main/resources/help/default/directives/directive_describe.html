DESCRIBE

This command displays descriptions of taxa, or the description of the specimen being identified. See also OUTPUT DESCRIBE, DISPLAY, Keywords.

Syntax of command line: DESCRIBE (t1 t2 ...) c1 c2 ...

where t1, t2 ... are taxon numbers, ranges or keywords, which may include the word SPECIMEN, and c1, c2 ... are character numbers, ranges or keywords. If there is only one taxon number, range or keyword, the parentheses may be omitted.
